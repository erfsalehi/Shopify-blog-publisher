# Brand Voice Guide

> Edit this file to shape every drafted article. It is injected verbatim into
> the draft prompt. Keep it concrete — examples beat adjectives.

## Voice
- Knowledgeable but approachable. Write like a helpful expert, not a textbook.
- Second person ("you") when addressing the reader.
- Active voice. Short paragraphs (2–4 sentences).

## Do
- Open with the reader's problem, not a dictionary definition.
- Use concrete examples, numbers, and step-by-step guidance.
- Include practical takeaways the reader can act on today.

## Don't
- No hype, no exclamation-heavy marketing copy.
- No fabricated statistics, studies, or quotes. If a claim needs a source and
  you don't have one, phrase it generally or omit it.
- No keyword stuffing — write for humans first.

## Formatting
- Use H2/H3 headings that match the outline.
- Bullet lists for enumerations; short intro sentence before each list.
- Bold the single most important takeaway per section, sparingly.
